# Projct_Costs
Este é um gerenciador de projetos.
Este projeto foi desenvolvido utilizado o React, a biblioteca JavaScript mais utilizada da atualidade.
O projeto tem como objetivo gerenciar projetos e seus custos, onde você poderá nomear o tipo de projeto e custos do projeto, e possuim a funcionalidade
ligth mode e dark mode.
## Como executo o projeto?
é só copiar link: http://localhost:5182/, no seu navegador e testar!
